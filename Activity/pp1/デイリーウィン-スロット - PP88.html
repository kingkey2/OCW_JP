ゲームズ

1. トーナメントの勝者は、最高シングルスピン勝利額 (ベット額に合わせて調整) に基づいています。​

​

2.  スコア = (トーナメント中の勝利額 (USD 換算))/(トーナメント中のベット額 (USD 換算)) * 1000​

サンプル計算:
プレーヤー 1: $1 を賭ける -> $50 を勝ち取る (スコア = $50 / $1 * 1000 = 50,000)
プレーヤー 2: $1 を賭ける -> $5 を勝ち取る (スコア = $5 / $1 * 1000 = 5,000)

​

3. このトーナメントは毎週開催され、 2022年7月28日 18:00から2022年8月25日 17:59（北京時間）まで、木曜日から木曜日にかけて行われます。プロモーション期間中に4つのトーナメントがあります

4. 参加するための最小ベットは、60円または同等額です。​​

​

5. 対象となるゲーム：Money Roll, Irish Charms, 888 Gold, Diamonds are Forever 3 lines，テーブルゲーム、ビデオポーカー、スクラッチカードを除くすべてのPragmatic Playゲーム。

​

6. トーナメントのリーダーボードに同じスコアのプレイヤーが2人以上いる場合、最初にそれを獲得したプレイヤーが当該リーダーボードでより高いポジションを獲得します。​​

​

7. トーナメントリーダーボードは対象ゲームで構築され、リアルタイムでアップデートされます。​​

​

8. Pragmatic Playは、予告なしにいつでもプロモーションを修正、一時停止、またはキャンセルする権利を留保いたします。​

ゲームズ

1. デイリーキャッシュドロップマルチプライヤーでベットの最大1,000倍を獲得できます​​

​

2. 2賞金マルチプライヤーを使用して、最終賞金額（ベット額 x 獲得マルチプライヤー）が決定されます。大きなベットを行うと、1回のスピンで最大575,000円のキャッシュを獲得するチャンスがあります​​

例：​​

ベット額130円​

獲得マルチプライヤー： X 500​​

獲得賞金：65,000円​

​

3. 賞金は、ベット額にマルチプライヤーを掛けて計算されます。賞金計算の対象となる最大ベット額は575円です。ベットが575円を超えた場合、を超えた追加額は賞金の計算に使用されません。​​

例：​​

ベット額：800円​

賞金計算の対象となるベット額：575円​

獲得マルチプライヤー： X 1000​​

獲得賞金: 575円  X 1000 = 575,000円

​

4. 参加するための最小ベットは、60円または同等額です​​

​

5. 2,990,000円相当の3500種類の賞があり、毎週プロモーション期間中に幸運なプレイヤーの方々に与えられます。​​

​
6. このトーナメントは毎週開催され、2022年7月28日 18:00から2022年8月25日 17:59（北京時間）まで、木曜日から木曜日にかけて行われます。

​

7.対象となるゲーム：Money Roll, Irish Charms, 888 Gold, Diamonds are Forever 3 lines, Table Games, Video Poker, Scratch cards, Great Reef, Busy Bee, Dwarven Gold and Devil’s 13を 除くすべてのPragmatic Playゲーム。

​

8. Pragmatic Playは、予告なしにいつでもプロモーションを修正、一時停止、またはキャンセルする権利を留保いたします。​
